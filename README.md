# Simple To-Do App

## Description

This project is a simple to-do application that allows users to add, complete, and delete tasks. The main focus of this project is organizing JavaScript code using classes and modules.

The project uses object-oriented programming to separate responsibilities between different parts of the app. The `Todo` class handles the creation and behavior of individual to-do items, while the `FormValidator` class handles form validation.

## Functionality

Users can:

- Add new to-do items
- Mark to-dos as completed
- Delete to-dos
- See form validation messages
- Submit the form only when the input is valid

## Project Structure

The JavaScript code is organized into separate class files:

- `Todo.js` contains the `Todo` class
- `FormValidator.js` contains the `FormValidator` class
- `index.js` imports and uses both classes

Each class is responsible for one specific task. This keeps the code cleaner, easier to understand, and easier to maintain.

## Project Goals

The goal of this project was to practice organizing code into reusable classes. Instead of keeping all the JavaScript in one file, the project separates responsibilities into different modules.

This helped reinforce:

- How to create JavaScript classes
- How to export classes from separate files
- How to import classes into `index.js`
- How to keep code organized by responsibility
- How to make a project easier to read and maintain
